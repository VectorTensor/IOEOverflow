IOEOVERFLOW

A peer reviewed question answer site for IOE engineering students.

{% block content %}

The title of our project is IOE Overflow. The aim of this project is to create a platform to share the answers of questions that you have practiced. Then it will be reviewed by other members. The main purpose of this system is to make the question answer flow between students easier. For this a solution is to construct a student website where students can enter their doubts and it can be answered by other students. The answers can then be reviewed by other students and the number of ratings helps to validate the given answer. We will be using frameworks for ease of access. Login can be made based on the student ID, which can further validate the reliability of the project. The answers given by the users can also be saved in their login module. The credibility of the user is also given by the number of good ratings that they get from their answers.
